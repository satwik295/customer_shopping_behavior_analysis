# Customer Shopping Behavior Analysis

## About the Project

This project analyzes customer shopping behavior to uncover useful business insights from retail data. It covers the complete data analytics workflow, from cleaning raw data to building an interactive dashboard for decision-making.

---

## Tools Used

- Python
- Pandas & NumPy
- SQL
- Power BI
- Gamma

---

## What I Did

- Cleaned and prepared the dataset
- Performed exploratory data analysis (EDA)
- Created new features for better analysis
- Wrote SQL queries to answer business questions
- Built an interactive Power BI dashboard
- Summarized key findings and recommendations

---

## Key Insights

The analysis highlights customer purchasing patterns, revenue trends, product performance, subscription behavior, and customer segments that can help businesses make better decisions.
